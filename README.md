# ライフプランOS

家計・資産のライフプランシミュレーター（日本の世帯向け）。

**公開URL**: https://subara1.github.io/lifeplan/

## 主な機能

- 90歳までのキャッシュフロー・資産推移シミュレーション（グラフ・CF表・ライフイベント表）
- 資産エンジン: 現役中は積立投資、現金不足時は投資資産を自動取り崩し
- 年金の詳細推計: 基礎+厚生を本人・配偶者別に自動推計、繰上げ/繰下げ・物価スライド対応
- 理想の生活費バランス: 家族構成と手取りから理想の家計配分を算出して現状と比較
- 支出入力欄に家族構成別の一般的な目安を表示
- シナリオ比較・退職時期比較・目標管理・AIコーチ（ルールベース）
- Excel/CSV/JSONエクスポート、入力値の自動保存

## 開発

- 構成: `index.html` 単一ファイル（詳細は `CLAUDE.md` / `AGENTS.md`）
- テスト: `node tests/run-tests.js`
- デプロイ: main へ push すると GitHub Pages に自動反映
- CI: push/PRごとにテスト・構文・描画スモークテストを実行
